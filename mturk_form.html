Survey Instructions

For each question, you are given a random fragement of a document from the internet. Please categorize the document into one of five buckets based on how typical it is as an internet text. Typicality is a measure of how frequently a fragment would be seen on the internet.

We have labeled a number of the obvious examples oursleves, but their labels are not shown. These serve as test questions and we will reject your submission if you are too far off from the truth.

DESCRIPTION OF BUCKETS

Very Typical: You expect to see this all the time.

Typical: You often expect to see something like this.

Average: Not surprised to see this, but would not appear as typically as the above.

Specific: This is a reasonable fragment, but only in a very specific setting.

Rare: There is some context where this is a reasonable text, but you would be surprised to see it.

Invalid: This is not human-readable text, so it is difficult or impossible to rate its typicality.
